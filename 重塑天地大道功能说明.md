# 重塑天地大道功能说明

## 功能概述
重塑天地大道是专为准仙帝级玩家（mijinglevel_id > 19）设计的强大功能，允许玩家以准仙帝的伟力改变秘境的奖励概率。

## 使用条件
- 玩家必须达到准仙帝境界（mijinglevel_id > 19）
- 必须在群聊中使用（私聊功能不开放）

## 指令格式
```
#重塑天地大道 [低级概率] [中级概率] [高级概率]
```

## 参数说明
- **低级概率**: 低级秘境的出金概率（0-1之间的数值）
- **中级概率**: 中级秘境的出金概率（0-1之间的数值）  
- **高级概率**: 高级秘境的出金概率（0-1之间的数值）

## 使用示例

### 示例1：提高所有秘境概率
```
#重塑天地大道 0.95 0.8 0.6
```
- 低级秘境概率：95%
- 中级秘境概率：80%
- 高级秘境概率：60%

### 示例2：降低高级秘境概率，提高低级秘境概率
```
#重塑天地大道 0.99 0.4 0.2
```
- 低级秘境概率：99%
- 中级秘境概率：40%
- 高级秘境概率：20%

### 示例3：平衡设置
```
#重塑天地大道 0.7 0.5 0.3
```
- 低级秘境概率：70%
- 中级秘境概率：50%
- 高级秘境概率：30%

## 默认配置
系统的默认概率设置为：
- 低级秘境：0.99 (99%)
- 中级秘境：0.5 (50%)
- 高级秘境：0.25 (25%)

## 注意事项
1. **权限限制**：只有准仙帝级玩家才能使用此功能
2. **参数范围**：所有概率值必须在0-1之间
3. **即时生效**：修改后的概率立即生效，影响所有后续的秘境探索
4. **全局影响**：修改会影响服务器内所有玩家的秘境探索结果
5. **持久化**：设置会保存到配置文件中，重启后仍然有效

## 相关配置文件
概率设置保存在 `config/xiuxian/xiuxian.yaml` 文件中的 `SecretPlace` 部分：

```yaml
SecretPlace:
  one: 0.99    # 低级秘境概率
  two: 0.5     # 中级秘境概率  
  three: 0.25  # 高级秘境概率
```

## 权衡考虑
作为准仙帝，在重塑天地大道时需要考虑：
- **游戏平衡**：过高的概率可能影响游戏平衡
- **玩家体验**：合理的概率设置能让所有玩家都有良好体验
- **长期影响**：修改会影响整个服务器的游戏环境

## 其他相关功能
- `#查看时代` 或 `#时代` - 查询当前时代状态
- `#突破模拟` - 模拟当前时代的突破成功率
- `#重塑古今未来 [时代索引] [年份] [纪元]` - 重塑时代（仙帝专属）
